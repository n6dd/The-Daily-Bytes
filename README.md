1. Clone the repository:
    ```bash
    git clone https://github.com/n6dd/The-Daily-Bytes.git
    ```

2. Navigate into the project directory:
    ```bash
    cd The-Daily-Bytes
    ```

3. Install the required dependencies:
    ```bash
    npm install
    ```

4. Run the app:
    ```bash
    npm run start

    npm run start:dev
    ```
